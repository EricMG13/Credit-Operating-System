## Rejected as non-credit (do NOT spec)

- News/RSS sentiment ticker — external-feed-bound, trust erosion (FEATURE_IDEATION §5.3). Automated transport rejected; pasted-intel extraction lane is separate + backend-gated.
- Positions / P&L / holdings — turns analysis platform into quasi-OMS, off-goal (§5.2). Sleeve membership is the ceiling.
- Realtime collaboration / presence / co-editing — single-team, deep-work personas (§5.6).
- Write-action copilot ("AI updates the model") — violates the no-LLM-writes safety property (§5.4). Read-only NL + RAG is the sanctioned shape.
- Bond analytics / STW / Z-spread — loans-only, DM-canonical (§5.7).
- Generic SaaS chrome: activity feeds, gamification, social, vanity KPIs — no credit-conclusion payoff (brief §1 anti-pattern).
